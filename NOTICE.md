# Notice

This repository publishes TEI editions and a static viewer for Dioscorides,
including a normalized Berendes edition and an initial diplomatic Sprengel
import.

Berendes source facsimile:

- Pedanius Dioscorides; Julius Berendes, trans., *Des Pedanios Dioskurides aus
  Anazarbos Arzneimittellehre in fünf Büchern* (Stuttgart: Verlag von Ferdinand
  Enke, 1902).
- Heidelberg University Library digitization:
  https://digi.ub.uni-heidelberg.de/diglit/berendes1902
- DOI: https://doi.org/10.11588/diglit.69903
- URN: urn:nbn:de:bsz:16-diglit-699033

Sprengel source facsimile:

- Pedanius Dioscorides; Curtius Sprengel, ed., *Pedanii Dioscoridis Anazarbei De
  materia medica libri quinque* (Lipsiae: Cnobloch, 1829/1830).
- Wellcome Collection record:
  https://wellcomecollection.org/works/ncazpf6u

The static viewer loads page images from Heidelberg facsimile URLs recorded in
the Berendes TEI and manifest. Sprengel image paths are prepared for later
Wellcome image/IIIF alignment, but page images are not bundled. This project
does not claim ownership of, bundle, or relicense external image files.
